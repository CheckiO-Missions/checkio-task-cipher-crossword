Все мы пробовали решить кроссворд (ну или хоть видели его). Давай добавим криптографии. В криптокроссворде вместо подсказок на линию используются подсказки на клетки. Клетки имеют номера от 1 до 26. Каждое число соответствует какой-то букве (и только одно) английского алфавита. Также тебе известны слова для кроссворда и твоя задача корректно его заполнить.

Дан пустой кроссворд, как 2-ух мерный {% if is_js %}массив{% else %}список{% endif %} с числами, где 0 - это пустая клетка, а остальные числа соответствуют зашифрованным буквам. Также дан {% if is_js %}массив{% else %}список{% endif %} слов для кроссворда. Вы должны заполнить кроссворд и вернуть решенный кроссворд, как 2-ух мерный {% if is_js %}массив{% else %}список{% endif %} с буквами. Пустые клетки замените пробелами (0 => " ").

Слова размещаются в строках и столбцах, но не в диагоналях. Кроссворд состоит из 6 слов, каждое из 5 букв.

Входные данные: Кроссворд, как {% if is_js %}массив массивов{% else %}список списков{% endif %} с целыми числами. Слова, как {% if is_js %}массив{% else %}список{% endif %} строк.

Выходные данные: Решение кроссворда, как {% if is_js %}массив массивов{% else %}список списков{% endif %} с буквами.
